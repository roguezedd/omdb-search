# OMDB Search Web App

* Created with node 8.
* View the app in a web browser at: http://localhost:3030/
* Webapp is isomorphic.
* Server side render a search by using queryparam 'search': http://localhost:3030/?search=star

## Install dependencies
~~~~
npm install
~~~~

## Run the dev server
~~~~
npm run watch
~~~~

## Run the server
~~~~
npm run start
~~~~
